PayCheck Project

This project is a simple C++ paycheck calculator that computes an employee's final earnings based on:
Years of service
Monthly sales
Bonus rules based on experience and performance

The program collects user input,applies the appropriate bonus formulas and outputs the final paycheck amount.
Features 
Input for years worked
Input for monthly sales
Calculates first and second bonuses
Computes the total paycheck
clear and simple console interface 

Technologies Used 
C++
Clion IDE
CMake
